## How to use
Before building and using this application there are a few standard pre-requisites for a NodeJS based web application.

### Pre-Requisite
1. Install NodeJS (and NPM) from https://nodejs.org/
2. Install Grunt

   `npm install -g grunt-cli`

3. Install Bower

   `npm install -g bower`
   

### Building

1. `npm install`
2. `bower install`
3. `grunt`
